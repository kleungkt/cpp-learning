13. Std::string

15. Function/Class Template(模版)

17. Object Oriented Programming(OOP，面向对象编程)
18. Static, Friend, Nested function/Class(静态、友元、嵌套)

20. Inheritance(继承)
21. Modern C++
22. STL
23. Common interview questions(面试常问问题)
